# Code of Conduct

## Expected Behavior

- Be respectful and constructive.
- Discuss technical disagreements with evidence.
- Credit prior work and third-party sources.
- Avoid harassment, discrimination, personal attacks or disruptive behavior.

## Unacceptable Behavior

- Publishing private information without permission
- Introducing secrets or unsafe instructions into examples
- Misrepresenting model readiness, certification or third-party authorship
- Harassment or abusive language

## Enforcement

Maintainers may remove disruptive content, close issues or restrict participation to protect the project. For serious concerns, use GitHub repository reporting features where available.
